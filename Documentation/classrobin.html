◆ generatePoint()

Vector3 robin.generatePoint ( )
inlineprivate

generatePoint() is called whenever we need to generate a new random end point for the GameObject to move towards.

It generates random numbers between 4 and -4 for both x and z axis, and then adds these to the current position of the Visitor GameObject to derive a new end point.
